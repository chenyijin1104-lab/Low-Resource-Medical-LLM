# Low-Resource-Medical-LLM
“从 Tensor 到医学智能体”——本仓库记录了我进入大厂医疗 AI 军团的进阶之路。通过在 RTX 4050 上复现 BioMedLM 的微调过程，深度拆解神经网络线性层原理与 Transformer 架构在医学文本挖掘中的应用。项目包含完整的 QLoRA 实验记录、医学知识库检索增强（RAG）构思以及针对临床工作流的轻量化部署方案。

> 📝 项目正在迭代中，首阶段侧重于医学数据集的清洗与 QLoRA 环境的搭建。
